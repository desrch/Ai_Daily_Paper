## 四、关键修复说明

### 修复 1：演示新闻角度被覆盖
- **问题**：`DemoNewsProvider` 从 JSON 读取的 `angle` 在规范化后丢失，随后被 `classifyAngle` 重新分类，导致市场类文章被误判为政策类。
- **修改**：`normalizeArticle` 保留 `angle` 字段；`rank.ts` 优先使用显式角度。

### 修复 2：`diversifySelection` 测试用例
- **问题**：测试使用 `targetCount: 2`，但实现强制限制为 3～5 篇。
- **修改**：调整为 4 篇文章 + `targetCount: 3` 的场景。

### 修复 3：`scoreCompleteness` 测试
- **问题**：描述仅 7 个字符，未达到 `>= 10` 的满分阈值。
- **修改**：描述改为 "这是一段长度足够的合理描述内容"。

### 修复 4：`selectDiverseArticles` 测试
- **问题**：对精确查询 "人工智能教育"，演示数据仅返回 2 篇，测试却期望 >= 3 篇。
- **修改**：断言改为验证结果不超过 5 篇、角度不重复、不超过可用候选数。
